TI-84 Plus CE Games

Download

Download and extract this zip file to a folder

Install

(You must have TI Connect CE installed as you will need it.) - Download it here

Step One:

Install a shell.

Your two choices are Cesium and Agame
The only difference is in Agame, you can set a passcode.

To install, locate the folder of the shell you want to install, and use TI Connect to copy all .8xp files to the calculator.

Step Two:

Install the libraries.

Use TI Connect to copy all the files in the clibraries folder to the calculator.

Step Three:

Install games.

All the games are located in their own folder in the Games folder.
Look at the readme for each game to install it.
